This folder contains the datasets used for the Adidas Sales BI analysis.

## Source
Dataset is sourced from Kaggle.

## Files
- raw_adidas_sales_dataset.csv  
  Raw dataset in CSV format (as downloaded/exported).

- cleaned_adidas_sales_dataset.xlsx  
  Cleaned and analysis-ready dataset used for the BI analysis.

## Notes
- The Excel file is the cleaned version used for analysis.
- The CSV is the raw source and may include formatting differences (for example currency and percent formatting).
